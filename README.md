# 🧠 Productivity Dashboard (Frontend Project)

A simple and clean **Productivity Dashboard** built using HTML, CSS, and JavaScript. This project helps users manage daily tasks, plan their day, stay motivated, and boost focus using a Pomodoro timer.

---

## 🚀 Features

### ✅ 1. To-Do List

* Add tasks with details
* Mark tasks as **important**
* Organize and track your daily work

### 📅 2. Daily Planner

* Plan your day efficiently
* Structure your schedule

### 💡 3. Motivation Section

* Displays motivational quotes
* Helps maintain focus and positivity

### ⏱️ 4. Pomodoro Timer

* Improve productivity using time blocks
* Focus sessions + break cycles

### 🎯 5. Daily Goals

* Set and track your daily goals
* Stay consistent and disciplined

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript (Vanilla)** – Functionality
* **Remix Icons** – Icons library
